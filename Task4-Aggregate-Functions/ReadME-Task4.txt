# Task 4 – Aggregate Functions and Grouping

## Goal
Practice using aggregate functions like SUM, COUNT, AVG and grouping data with GROUP BY and HAVING clauses.

## What I did
- Counted total books and unique departments  
- Calculated average published year and sum of book IDs  
- Grouped students by department and counted them  
- Used HAVING to filter groups with more than 1 student  
- Found max book ID per author  
- Counted distinct students per department  

## Tools
- MySQL Workbench 

## Outcome
- Learned how to summarize and analyze data using SQL  
- Practiced grouping and aggregate functions  
- Used Telangana-related examples for authors, students, and books
