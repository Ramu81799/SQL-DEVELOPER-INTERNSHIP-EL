# Task 3 – SQL SELECT Queries

## Goal
Learn how to get data from tables using SELECT queries. This includes filtering, sorting, and limiting results.

## What I did
- Selected all columns and specific columns from tables  
- Filtered students by department and checked emails  
- Used LIKE and BETWEEN to find specific data  
- Sorted books by year (both ascending and descending)  
- Limited output to a few rows  
- Found unique departments using DISTINCT  
- Used aliases for better column names

## Tools
- MySQL Workbench

## Outcome
- Practiced writing SELECT queries  
- Learned how to filter, sort, limit, and rename columns  
- Used Telangana-related examples for books and students
